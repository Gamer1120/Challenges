1)	(Windows) start WlanScanner.exe.
	(Linux) start wlanScan: "java wlanScan.wlanScan name" where name is the name of your wireless interface.
	(Both programs can be kept running, they do not perform any action unless a client connects)

2) 	Change your group name and start the java code.
